# Amazon Scraper Backend - 部署指南

## 方式1: 本地运行
```bash
pip install -r requirements.txt
python scraper.py
```
启动后后端地址为: `http://localhost:8080`

## 方式2: 部署到 Render.com（免费）

1. 在 GitHub 创建一个新仓库，把 `backend` 文件夹里的文件上传上去：
   - `scraper.py`
   - `requirements.txt`
   - `Procfile`
   - `render.yaml`

2. 打开 https://render.com ，用 GitHub 账号登录

3. 点击 **New** → **Web Service**

4. 选择你刚创建的仓库

5. 设置：
   - **Name**: `amazon-scraper`（随意）
   - **Runtime**: `Python`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn scraper:app --bind 0.0.0.0:$PORT --workers 2 --timeout 30`
   - **Plan**: `Free`

6. 点击 **Create Web Service**，等待部署完成

7. 部署成功后，Render 会给你一个地址，例如：
   `https://amazon-scraper-xxxx.onrender.com`

8. 把这个地址填入前端页面的「后端地址」设置中

## 注意事项
- Render 免费版服务会在 15 分钟无请求后休眠，首次访问可能需要等待 30-60 秒唤醒
- 爬虫请求可能被亚马逊限速，建议适度使用
